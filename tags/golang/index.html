Nest - An Implementation of the Nest API in Go

2019-03-22
#go  #golang  #nest  #development 
One of the first major open-source projects I’ve worked on in Go was to unify all of my home devices under the Home app provided by Apple’s iOS. I came to realize after acquiring many devices that there was just no universal solution to integrate these into Home. After doing some research and nifty network sniffing, most devices either had a published or unpublished RESTful API that I could implement along with Apple’s HAP specification.
